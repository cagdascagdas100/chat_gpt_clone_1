# ChatGPT Takip Promptları

## Kaynak kanıt dosyası doldurma promptu
Aşağıdaki kaynak için `source_evidence_template.json` dosyasını doldur. Sadece resmi kaynak sayfasındaki bilgileri kullan. Kaynak URL'sini, erişim tarihini, kapsamını, sınırlılıklarını ve gerekli QA alanlarını yaz.

## Metodoloji doğrulama promptu
Aşağıdaki metodoloji taslağını kontrol et. Polis/asayiş güvenliği için hangi kısımlar kanıtlı, hangi kısımlar varsayım, hangi kısımlar review gerektiriyor ayır.

## QA raporu promptu
Aşağıdaki run manifest ve çıktı sayılarını kontrol et. Bad score range, missing core fields, review count, no-downgrade ihlali ve source evidence completeness durumunu özetle.
