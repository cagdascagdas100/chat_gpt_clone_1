# deutsch_tests_1

Bu klasör C1/C2 Almanca kelime, kalıp ifade, gramer ve yazım odaklı interaktif test sayfası içerir.

## Dosya

- `index.html`: PC ve telefonda tarayıcıdan açılabilen tek sayfalık test uygulaması.

## Kullanım

1. GitHub Pages aktifse web adresinden açılır.
2. Değilse dosya GitHub'dan indirilip herhangi bir tarayıcıda açılabilir.
3. Test sonunda yanlışlar, doğru cevaplar ve hata kategorileri otomatik raporlanır.
4. `Bericht` düğmesiyle rapor `.txt` olarak indirilebilir.

## Yeni kelime listesi workflow'u

Bana yeni Nomen / Verb / Kalıp ifade listesini verdiğinde, bu klasördeki `index.html` dosyasını sadece o yeni listeye göre güncelleyeceğim. Eski konu kelimelerini yeni testlere karıştırmayacağım.
