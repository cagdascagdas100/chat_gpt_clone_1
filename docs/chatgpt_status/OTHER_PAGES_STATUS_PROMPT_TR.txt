Güncellenecek JSON alanları:
active_task: şu anki current-task id veya aktif iş adı
status: running / polling / finished / finished_with_failures / blocked / waiting
overall_progress: ana plana göre tahmini genel proje yüzdesi
wait_minutes: kullanıcının beklemesi gereken süre
next_command: devam et
runner_status: heartbeat içindeki runner durumu
runner_message: heartbeat mesajı veya exit kodu
blocker: varsa kısa blokaj açıklaması
last_message_text: kullanıcının son mesajı, genelde “devam et”
last_message_at: son kullanıcı mesajının tarih saati, format yyyy-MM-dd HH:mm:ss
db_write: false
production_deploy: false
updated_at: güncel tarih saat

Mümkünse şu helper script’i kullan:
ai-task-scripts/update_chatgpt_status.ps1

Örnek PowerShell helper çağrısı:
.\ai-task-scripts\update_chatgpt_status.ps1 `
  -PageKey "[SAYFA_ADI]" `
